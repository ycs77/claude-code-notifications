# Claude Code 提示音合集

這裡是 Claude Code 提示音合集倉庫，會收錄各種有趣好玩的提示音插件。

## 插件列表

| Name | Sources |
|------|---------|
| 基本 | [Windows](./plugins/basic-win) \| [macOS](./plugins/basic-mac) |

## 在 Claude Code 中安裝

在 Claude Code 中執行以下命令，將此 repo 註冊為插件市場：

```
/plugin marketplace add ycs77/claude-code-notifications
```
